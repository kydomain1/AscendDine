# Ascend & Dine

A minimalist comprehensive blog website featuring content across six categories: Fashion & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, and Food & Beverage.

## Features

- **Minimalist Design**: Clean, focused interface with ample white space and limited color palette
- **6 Content Categories**: Diverse topics covering lifestyle, health, travel, finance, and more
- **Blog Articles**: 5 featured articles with rich content and authentic imagery
- **Product Recommendations**: Detailed product reviews and platform recommendations
- **Dual Search System**: Quick search on homepage + dedicated search page with advanced features
- **Category Filtering**: Easy navigation through 6 content categories
- **Pagination**: Organized content display with page navigation
- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Social Media Integration**: Connected social media icons
- **Smooth Animations**: Elegant transitions and interactions

## Structure

- `index.html` - Homepage with blog articles grid
- `search.html` - Dedicated search page with advanced filtering
- `article.html` - Individual article detail pages
- `about.html` - About page
- `contact.html` - Contact page
- `css/style.css` - Main stylesheet
- `js/main.js` - JavaScript functionality
- `images/` - Image assets

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Responsive Design

## Design Philosophy

**Minimalism**: Clean, clear, focused, breathing space, function-first approach
- Large white space
- Limited color palette
- Simple lines
- Removal of all unnecessary elements

## Getting Started

Simply open `index.html` in a web browser to view the website.
